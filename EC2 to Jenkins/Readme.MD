![add storage](./EC2-to-Jenkins-images/step-1-add-storage.jpg)

![Add tag/name](./EC2-to-Jenkins-images/step-1-add-Tag.jpg)

![Add security Group](./EC2-to-Jenkins-images/step-1-add-security-group.jpg)
